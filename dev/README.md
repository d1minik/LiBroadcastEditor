# LiBroadcastEditor

LiBroadcastEditor is a Chrome extension for customizing Lichess broadcast pages.

## Installation

1. Download the ZIP file that contains the extension code files.
2. Extract the ZIP so all files are inside one folder.
3. Open Chrome and go to `chrome://extensions/`.
4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted folder that contains `manifest.json`, `popup.html`, `popup.js`, `content.js`, and `content.css`.

## Optional Starter Template

If you want a quick starting point, download the separate `template_v0.1.css` file too.

1. Open the LiBroadcastEditor popup in Chrome.
2. Click **Import CSS**.
3. Select `template_v0.1.css`.
4. The template settings will be imported into the extension.
